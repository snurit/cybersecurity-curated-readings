# cybersecurity Curated Readings
Une liste de sites internet, articles et autres ressources concernant la cybersecurité.
La pluspart de ces ressources sont gratuites ou nécessitent la création d'un compte utilisateur gratuit offrant un accès à des fonctionnalités limitées mais utiles.

## Catégories
- [GRC](./GRC) : Gouvernance, Risques et Conformité
- [OSINT](./OSINT) : Open Source INTelligence
- [Blue Teaming](./BLUE_TEAMING/) : Ressources et outils défensifs
- [Red Teaming](./RED_TEAMING/) : Ressources et outils offensifs
